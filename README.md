# Project-UAS-Kel-5
Sistem Inventory merupakan aplikasi berbasis web yang digunakan untuk membantu proses pengelolaan persediaan barang pada suatu gudang atau toko. Sistem ini dirancang untuk memudahkan admin dalam melakukan pencatatan data barang, kategori, supplier, customer, serta transaksi barang masuk dan barang keluar secara terkomputerisasi.
